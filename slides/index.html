3 Logistical Problems


Challenge:
Solve 3 logistical problems by developing an algorithm.

Shortest Path

Find the shortest path between two given cities.

Salesman Route

Find the optimal route for visiting all cities once.

Trade Route

Find the most profitable route between cities within a maximum given action window.

Shortest Path

  • Terrain broken up in a grid
  • Different movement cost for each tile

A*

  • Easy to use
  • Highly efficient
  • For grid-based terrains
  • Widely used

How does it work?

Salesman Route

  • Must start/end in one of the cities
  • Must visit all cities once
  • Take movement cost into account
  • Must calculate in reasonable time frame

Genetic Algorithm

  • Optimal results in minimum time
  • Simple concept
  • Always an answer, gets better over time

  • Natural selection, following Darwin's theory

How does it work?

  • Create population
  • Individuals try random routes

  • Determine individual fitness
  • New generation
  • Crossover from fittest
  • Mutation from fittest
  • Fittest survive, others die

Trade Route

  • Find most profitable trade plan/route
  • Start in specified city
  • Within specified number of actions
  • Must calculate in reasonable time frame

Custom Algorithm

  • Optimal results in minimum time
  • Always an answer, gets better over time

  • Uses highly modified genetic algorithm
  • Implemented machine learning

How does it work?

  • Population based like genetic algorithm
  • Multiple trade tactics are considered
  • Individuals learn optimal routes and trade tactics
  • Learned routes are preferred by individuals
  • Knowledge is taught to new generations
